== Description ==

Simple paypal button visual composer addon plugin.

Simple paypal button visual composer works like any standard WordPress plugin, and once installed and enabled, it will automatically added a paypal button to your website which has visual composer addon plugin installed and enabled.

Simple Paypal Button is a Visual Composer plugin that allows for creation of Paypal "Buy Now" buttons.

It includes the following features:
 <ul>
   <li>Drag-and-Drop Paypal Button Creation</li>
   <li>Support for Custom Button Images</li>
   <li>Add custom Shipping and Flat Tax Fees</li>
   <li>Option for opening a New Tab when button is clicked</li>
   <li>Multicurrency Support</li>
   <li>Sandbox Mode Support</li>
 </ul>
   
You can find documentation for installing this plugin here.
http://oliveconcepts.com/paypal-button-visual-composer

== Installation ==
This section describes how to install the plugin and get it working.

Simple paypal button Edition works like any standard Wordpress plugin, and once enabled will automatically add a paypal button to your visual composer addon enabled website.

= Installing =
1. If installing, go to Plugins > Add New in the admin area, and search for 'simple paypal button for visual composer'.
2. Click install, once installed, activate and you're done!
